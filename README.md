# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## How it works
The script takes:
- Principal amount
- Rate of interest
- Time period

Then it calculates and displays the simple interest.

## Example
If:
Principal = 1000  
Rate = 5  
Time = 2  

Simple Interest = 100
